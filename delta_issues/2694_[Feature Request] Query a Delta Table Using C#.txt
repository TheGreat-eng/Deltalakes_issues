## Feature request

### Overview

As a developer working with Delta Lake I would like to be able to query a delta table using C# so that I do not have to rely on Apache Spark to retrieve my data. 

### Motivation
This feature will be used to reduce costs by allowing applications to query a delta table directly instead of relying on Spark.
