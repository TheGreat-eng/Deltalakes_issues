### Overview
Add support for `IS_NULL` expression. The semantics of the expression are similar to SQL (examples [here](https://www.w3schools.com/sql/sql_isnull.asp))

[Here](https://github.com/delta-io/delta/pull/2360) is an example PR that added the support for `IS_NOT_NULL` which can be used as template to add the support for `IS_NULL`.
