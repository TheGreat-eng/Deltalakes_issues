## Protocol Change Request

### Description of the protocol change

This protocol change adds support for Variant shredding for the Variant data type.
Shredding allows Variant data to be be more efficiently stored and queried.

The [Parquet Variant Shredding specification](https://github.com/apache/parquet-format/blob/master/VariantShredding.md) describes the details of how Variant values are shredded in Parquet.
