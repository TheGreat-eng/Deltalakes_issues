## Feature request

### Overview

Delta Lake has very nice design for lakehouse, but it needs some extension mechanism, such as extension for building index.

### Motivation

There are a lot of unstructure data like image, text, audio and video. In general, user store them in vector with embedding. So we  to build Faiss or Annoy index on vector column to accelerate the queries.
