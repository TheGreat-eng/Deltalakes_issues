### Description of the protocol change
This protocol change is part of the proposed Type Widening table feature, see feature request: https://github.com/delta-io/delta/issues/2622

Design doc: https://docs.google.com/document/d/1KIqf6o6JMD7e8aMrGlUROSwTfzYeW4NCIZVAUMW_-Tc

The type widening table feature covers changing the type of existing columns or nested fields in a Delta table without having to rewrite the data.
