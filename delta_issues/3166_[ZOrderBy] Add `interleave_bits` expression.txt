It implements `interleave_bits(col1:int, col2:int, â€¦. col_n:int) -> byte array (Z-order value)`. This expression is used to combine multiple Z-order by columns into a [Z-order value](https://en.wikipedia.org/wiki/Z-order_curve). This Z-order value is used to layout the data such that the records with Z-Order By column having close values remain close when stored in files.
