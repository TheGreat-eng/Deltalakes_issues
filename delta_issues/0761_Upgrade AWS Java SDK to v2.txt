### Overview

<!-- Provide a high-level description of the feature request. -->

The AWS Java SDK v1 has entered maintenance mode July 31 2024 (https://aws.amazon.com/blogs/developer/announcing-end-of-support-for-aws-sdk-for-java-v1-x-on-december-31-2025/)

AWS Java SDK v2 has been around for a while now, and is generally better structured than the v1 sdk. It would be good to upgrade this dependency in the delta libraries to the v2 version.


### Motivation

<!-- How will this feature be used? Why is it important? Which users will benefit from it? -->

This upgrade is important for security and performance. All customers should benefit from it.
